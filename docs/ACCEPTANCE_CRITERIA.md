# 第一版驗收標準

## 帳號

- [x] 學生不能只知道別人的學號就搶先建立密碼。
- [x] 一次性啟用碼使用後不能再次使用。
- [x] 任課老師可自行註冊，但核准前看不到任何班級資料。
- [x] 導師可核准、停用教師並設定任教科目。
- [x] 導師可重設學生密碼。

## 權限

- [x] 數學老師不能查看或修改英語的私密作業管理資料。
- [x] 小老師只能操作被指派科目，不能進入系統設定。
- [x] 學生／家長只能查看自己的缺交資料；成績介面留待後續版本。
- [x] 服務端拒絕越權請求，不能只靠隱藏按鈕。

## 分組作業

- [x] 支援數學 A／英語 A。
- [x] 支援數學 A／英語 B。
- [x] 支援數學 B／英語 A。
- [x] 支援數學 B／英語 B。
- [x] A 組學生看不到 B 組作業，反之亦然。
- [x] 所有學生都能看到共同作業。
- [x] 調整學生新分組後，舊作業對象不改變。

## 繳交與補交

- [x] 全班皆交時可用一個按鈕完成點收。
- [x] 有例外時只需選取異常學生。
- [x] 小老師登記後立即生效。
- [x] 任課老師只能修正本科紀錄；導師可修正全部。
- [x] 請假及公假可設定下一次上課追繳期限。
- [x] 追繳期限過後，可轉為未完成、未攜帶或遲交並開始計數。
- [x] 免交不計入未繳及遲交。
- [x] 已補交紀錄 3 天後從待辦區消失，但仍存在歷史與統計。
- [x] 同一作業的遲交次數最多計一次。
- [x] 本週、本月、本學期及本學年統計同時可見。

## 公告與榮譽榜

- [x] 導師可發布全校公告與班級公告。
- [x] 公告可包含圖片。
- [x] 學生可標記已讀，導師可查看已讀及未讀名單。
- [x] 榮譽榜向全班顯示姓名、事蹟及日期。
- [x] 榮譽榜不顯示個人成績或缺交統計。

## 介面與部署

- [ ] 手機可完成登入、查看作業、公告已讀及教師點收。
- [x] 介面採用可愛但適合國中生的 3D 卡片風格。
- [x] 有「前往各科學習系統」按鈕。
- [x] 學生可用獨立標籤切換聯絡簿與班級行事曆。
- [x] 行事曆可切換月份、查看跨日行程與當日詳細內容。
- [x] 導師可新增、編輯、下架及恢復行程，學生與任課老師只能查看。
- [x] GitHub Pages 最新部署為成功狀態（2026-07-21）。
- [ ] 正式資料匯入前已完成所有角色的權限測試。
